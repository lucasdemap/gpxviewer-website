GPX File Not Opening?

Fix corrupted downloads, wrong formats, and import errors.

July 16, 2026

If a GPX file will not open, the cause is usually a bad download, wrong file type, or no app assigned to .gpx.

Quick fixes

  • Confirm the file ends in .gpx (not .xml or .kml)
  • Re-download or re-export from Strava, Komoot, or the source
  • Open with GPX Viewer via Share → Open with
  • Check the file is not empty (0 KB)

Wrong format? Compare GPX vs KML and GPX vs TCX.
