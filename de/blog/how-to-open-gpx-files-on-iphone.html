GPX-Dateien auf iPhone und Android öffnen

GPX-Routen aus Mail, Dateien oder Safari direkt in GPX Viewer importieren.

11. Juli 2026

Sie haben online einen tollen Wanderweg gefunden, ein Freund hat Ihnen eine Radroute geschickt oder Sie haben eine GPS-Aufzeichnung exportiert — jetzt müssen Sie die GPX-Datei auf Ihrem Handy öffnen. Die gute Nachricht: GPX auf iPhone und Android zu importieren ist unkompliziert, sobald Sie wissen, welches Teilen-Menü Sie nutzen müssen.

Dieser Ratgeber führt Sie durch jeden gängigen Importweg und zeigt, wie GPX Viewer Ihre Route innerhalb von Sekunden auf einer Karte anzeigt. Wenn Sie mit dem Format selbst noch nicht vertraut sind, beginnen Sie mit was ist eine GPX-Datei für Hintergrund zu Tracks, Routen und Wegpunkten.

Vor dem Import: einen GPX Viewer installieren

iOS und Android liefern keinen eingebauten GPX-Kartenviewer mit. Sie brauchen eine App, die das .gpx-Format versteht. GPX Viewer ist kostenlos auf beiden Plattformen verfügbar und übernimmt Import, Kartenanzeige, Höhenprofile und Routenerstellung.

  • Laden Sie GPX Viewer aus dem App Store oder Google Play herunter.
  • Öffnen Sie die App einmal, damit sie sich als GPX-Handler auf Ihrem Gerät registriert.
  • Halten Sie die GPX-Datei bereit — in Mail, Dateien oder Ihrem Download-Ordner.

GPX-Datei auf dem iPhone öffnen

Apples Teilen-Blatt ist der schnellste Weg, GPX auf dem iPhone zu importieren. Die genaue Bezeichnung variiert je nach Speicherort der Datei leicht, aber der Ablauf ist immer: Datei finden → Teilen tippen → GPX Viewer wählen.

Aus der Mail-App

  1. Öffnen Sie die E-Mail mit dem .gpx-Anhang.
  2. Tippen Sie auf den Anhang zur Vorschau.
  3. Tippen Sie auf das Teilen-Symbol unten links.
  4. Scrollen Sie in der App-Zeile und wählen Sie GPX Viewer.
  5. Die Route erscheint sofort auf der Karte.

Aus der Dateien-App

  1. Finden Sie die .gpx-Datei in iCloud Drive, Auf meinem iPhone oder einem verbundenen Cloud-Dienst.
  2. Halten Sie die Datei gedrückt und tippen Sie auf Teilen, oder öffnen Sie sie und tippen Sie auf das Teilen-Symbol.
  3. Wählen Sie GPX Viewer aus der App-Liste.

Aus Safari

Viele Wanderwebsites bieten einen direkten GPX-Download-Link. Tippen Sie auf den Link, dann auf Herunterladen. Wenn der Download abgeschlossen ist, tippen Sie im Safari-Download-Manager darauf und teilen Sie ihn mit GPX Viewer. Sie können die Datei auch zuerst in Dateien speichern und von dort importieren.

Aus einer heruntergeladenen GPX-Datei

Wenn Ihnen jemand eine Route geschickt hat oder Sie eine .gpx von einer Website gespeichert haben, öffnen Sie die Datei und tippen Sie auf Teilen. Wählen Sie GPX Viewer aus der Liste. Die Route erscheint sofort auf der Karte — ohne Konvertierung oder zusätzliche Schritte.

GPX-Datei auf Android öffnen

Der Öffnen-mit-Dialog von Android funktioniert ähnlich wie das Teilen auf iOS, die Menüs sehen je nach Hersteller-Oberfläche jedoch unterschiedlich aus.

Aus Gmail oder einer E-Mail-App

  1. Öffnen Sie die Nachricht mit dem GPX-Anhang.
  2. Tippen Sie auf den Anhang zum Herunterladen.
  3. Wenn Sie dazu aufgefordert werden, wählen Sie GPX Viewer als öffnende App.
  4. Erscheint keine Aufforderung, öffnen Sie Ihren Download-Ordner und tippen Sie auf die Datei.

Aus lokalem Speicher oder Cloud-Ordner

  1. Finden Sie die .gpx-Datei im Dateimanager Ihres Handys oder in einem Cloud-Speicherordner.
  2. Tippen Sie auf die Datei, dann wählen Sie Öffnen mit oder das Drei-Punkte-Menü → Teilen.
  3. Wählen Sie GPX Viewer aus der App-Liste.

Aus dem mobilen Browser

Laden Sie die GPX von einer Website herunter und öffnen Sie dann die Download-Liste Ihres Browsers. Tippen Sie auf die fertige Datei und wählen Sie GPX Viewer. Wenn Android fragt, ob Sie die Datei nur einmal oder immer mit GPX Viewer öffnen möchten, wählen Sie Immer für schnellere Imports später.

GPX-Importprobleme beheben

Die meisten Importfehler haben wenige vorhersehbare Ursachen. Gehen Sie diese Checkliste durch, bevor Sie annehmen, die Datei sei defekt:

  • Falscher Dateityp — bestätigen Sie, dass die Endung .gpx ist, nicht .kml oder .tcx.
  • Leerer Track — manche Dateien enthalten Metadaten, aber keine GPS-Punkte; versuchen Sie, die Route aus GPX Viewer neu zu exportieren.
  • App nicht in Teilen aufgeführt — öffnen Sie GPX Viewer zuerst und versuchen Sie es erneut; starten Sie das Handy bei Bedarf neu.
  • Sehr große Datei — mehrtägige Aufzeichnungen mit sekündlicher GPS-Abtastung können langsam sein; warten Sie einige Sekunden auf die Verarbeitung.

Öffnet sich die Datei, sieht aber auf der Karte falsch aus, lesen Sie unseren Ratgeber GPX auf einer Karte anzeigen, um Kartenstil, Zoomstufe und Höheneinstellungen zu prüfen.

Was passiert nach dem Import?

Sobald GPX Viewer Ihre Datei geladen hat, sehen Sie die vollständige Route auf einer interaktiven Karte. Wischen Sie nach oben oder tippen Sie auf die Routenkarte, um Distanz, Höhengewinn und Wegpunktdetails zu sehen. Wechseln Sie zwischen Standard-, Satelliten- und Hybrid-Kartenebenen — je nachdem, ob Sie Waldwege oder offene Straßen erkunden.

Mit GPX Viewer Pro können Sie außerdem den Live-Standort aktivieren, um Ihre Echtzeitposition und Richtung auf dem Track zu sehen — unverzichtbar beim Folgen von GPX-Wanderwegen oder der Navigation einer geplanten Radroute.

Mehrere GPX-Dateien organisieren

GPX Viewer behält importierte Routen in Ihrer Bibliothek, damit Sie zwischen ihnen wechseln können, ohne erneut zu importieren. Benennen Sie Routen beim Speichern, gruppieren Sie nach Aktivitätstyp und löschen Sie alte Dateien, die Sie nicht mehr brauchen. Wenn Sie eine Route wieder teilen möchten, nutzen Sie die Exportfunktion, um eine frische .gpx-Datei für Freunde zu erzeugen.
