Was ist eine GPX-Datei? Der komplette Ratgeber für Wanderer & Radfahrer

Verstehen Sie das GPS Exchange Format und warum .gpx-Dateien die Outdoor-Navigation antreiben.

11. Juli 2026

Wenn Sie schon einmal einen Wanderweg heruntergeladen, eine Radroute von einem Freund erhalten oder eine GPS-Aufzeichnung von einem Handy oder einer Uhr exportiert haben, sind Sie fast sicher auf eine GPX-Datei gestoßen. Die Endung .gpx steht für GPS Exchange Format — einen offenen Standard, mit dem Geräte und Apps Standortdaten so teilen können, dass jedes kompatible Tool sie lesen kann.

Anders als proprietäre Kartenformate, die an eine Marke gebunden sind, wird das GPX-Dateiformat auf Smartphones und Geräten weit verbreitet unterstützt. Genau diese Portabilität ist der Grund, warum Wanderer, Radfahrer und Trailrunner es täglich nutzen.

Was enthält eine GPX-Datei?

Im Kern ist eine GPX-Datei ein XML-Dokument, das geografische Koordinaten zusammen mit optionalen Metadaten speichert. Die meisten Dateien enthalten eines oder mehrere dieser Elemente:

  • Tracks — ein durchgehender Pfad aus GPS-Punkten, oft während einer Wanderung oder Tour aufgezeichnet.
  • Routen — ein geplanter Weg mit wichtigen Abzweigungen, nützlich für turn-by-turn-Navigation.
  • Wegpunkte — einzelne Punkte wie Gipfel, Campingplätze oder Parkplätze.
  • Höhendaten — Höhe über dem Meeresspiegel an jedem Punkt, für Höhenprofile.
  • Zeitstempel — wann jeder Punkt aufgezeichnet wurde, hilfreich für Tempoanalysen.

Da das Format textbasiert und menschenlesbar ist, können Sie eine kleine GPX-Datei in jedem Texteditor öffnen und Breiten- und Längengrad sowie Höhenwerte Zeile für Zeile sehen. Für den Alltag rendert eine spezialisierte App wie GPX Viewer diese Rohdaten als interaktive Karte.

GPX-Tracks vs. GPX-Routen

Die Begriffe werden oft synonym verwendet, aber es gibt einen wichtigen Unterschied. Ein Track ist typischerweise etwas, das Sie bereits zurückgelegt haben — Ihr Handy oder Ihre Uhr hat jeden GPS-Fix unterwegs protokolliert. Eine Route wird meist im Voraus geplant und verbindet Wegpunkte, die zeigen, wohin Sie gehen möchten.

Wann Wanderer GPX-Tracks nutzen

Wanderverbände, Reiseführer und Mitwanderer teilen GPX-Tracks, damit Sie einem etablierten Pfad durch Wälder, alpine Landschaften oder Küstenwege folgen können. Die Datei auf dem Handy zu laden, gibt Ihnen eine visuelle Referenz, auch wenn Wegmarkierungen spärlich sind. Unser Ratgeber zum besten GPX Viewer fürs Wandern erklärt, worauf Sie beim Folgen von Wegen unterwegs achten sollten.

Wann Radfahrer GPX-Routen nutzen

Radfahrer planen Rundkurse, Sportives und Gravel-Abenteuer als GPX-Routen und laden sie auf ein Handy oder einen Radcomputer. Wenn Sie eigene erstellen möchten, lesen Sie wie man GPX-Radrouten erstellt für einen Schritt-für-Schritt-Workflow.

Warum das GPX-Dateiformat zum Standard wurde

Vor GPX war das Teilen von GPS-Daten zwischen Marken mühsam. Jeder Hersteller nutzte sein eigenes Binärformat, und die Konvertierung erforderte spezielle Software. Das GPS Exchange Format löste das, indem es ein offenes XML-Schema veröffentlichte, das jeder implementieren kann.

Da GPX ein offener Standard ist, können Sie eine per E-Mail erhaltene Datei direkt in GPX Viewer auf Ihrem Handy öffnen — ohne Konvertierungsschritt.

Häufige Fragen zu GPX-Dateien

Ist GPX dasselbe wie KML?

Nein. KML ist ein anderes Kartenformat. GPX ist die bevorzugte Wahl für GPS-Navigation und das Folgen von Outdoor-Routen. GPX Viewer öffnet .gpx-Dateien direkt auf iPhone und Android.

Wie groß sind GPX-Dateien?

Ein kurzer Spaziergang in der Nachbarschaft kann eine 50-KB-Datei erzeugen, während eine mehrtägige Tour mit häufiger GPS-Abtastung mehrere Megabyte erreichen kann. Höhen- und Zeitstempeldaten erhöhen die Größe, aber auch große Dateien öffnen sich in modernen mobilen Apps schnell.

Kann ich eine GPX-Datei bearbeiten?

Ja. In GPX Viewer können Sie Routen von Grund auf erstellen, Wegpunkte bearbeiten und sie als neue .gpx-Dateien exportieren, um sie mit Freunden zu teilen oder später erneut zu öffnen.
