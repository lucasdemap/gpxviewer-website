FAQ

Quali tipi di file sono supportati?

GPX Viewer funziona con file .gpx — il formato standard per percorsi e tracce GPS.

C'è un abbonamento?

No. GPX Viewer Pro è un acquisto una tantum. Compra una volta, tienilo per sempre.

Posso aprire file GPX da altre app?

Sì. Condividi un file GPX con GPX Viewer da File, Mail, Safari e altre app.
