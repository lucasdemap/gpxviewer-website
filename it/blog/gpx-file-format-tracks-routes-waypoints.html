Tracce GPX, percorsi e waypoint

I tre elementi costitutivi all'interno di ogni file .gpx.

17 luglio 2026

Ogni file GPX può contenere tracce (registrazioni GPS), percorsi (itinerari pianificati) e waypoint (punti nominati come vette o parcheggi).

Cosa significa ciascuno

  • Traccia — percorso GPS registrato, il più comune da condividere
  • Percorso — sequenza di punti pianificata, a volte senza dettagli altimetrici
  • Waypoint — un luogo nominato sulla mappa
