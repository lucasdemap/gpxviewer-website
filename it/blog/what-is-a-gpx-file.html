Cos'è un file GPX? La guida completa per escursionisti e ciclisti

Comprendi il formato GPS Exchange e perché i file .gpx alimentano la navigazione outdoor.

11 luglio 2026

Se hai mai scaricato un sentiero escursionistico, ricevuto un percorso ciclistico da un amico o esportato una registrazione GPS da telefono o orologio, quasi certamente hai incontrato un file GPX. L'estensione .gpx sta per GPS Exchange Format — uno standard aperto che permette a dispositivi e app di condividere dati di posizione in un modo leggibile da qualsiasi strumento compatibile.

A differenza dei formati cartografici proprietari legati a un solo marchio, il formato file GPX è ampiamente supportato su telefoni e dispositivi. Questa portabilità è esattamente il motivo per cui escursionisti, ciclisti e trail runner ne fanno uso ogni giorno.

Cosa contiene un file GPX?

In sostanza, un file GPX è un documento XML che memorizza coordinate geografiche insieme a metadati opzionali. La maggior parte dei file include uno o più di questi elementi:

  • Tracce — un percorso continuo composto da punti GPS, spesso registrato durante un'escursione o una pedalata.
  • Percorsi — un tragitto pianificato con svolte chiave, utile per la navigazione passo dopo passo.
  • Waypoint — punti di interesse individuali come vette, campeggi o aree di parcheggio.
  • Dati altimetrici — altitudine sopra il livello del mare in ogni punto, usata per i profili di salita.
  • Timestamp — quando ogni punto è stato registrato, utile per l'analisi del ritmo.

Poiché il formato è basato su testo e leggibile dall'uomo, puoi aprire un piccolo file GPX in qualsiasi editor di testo e vedere latitudine, longitudine e altitudine elencate riga per riga. Per l'uso quotidiano, però, un'app dedicata come GPX Viewer rende quei dati grezzi una mappa interattiva.

Tracce GPX vs. percorsi GPX

Spesso si usano i termini in modo intercambiabile, ma c'è una distinzione utile. Una traccia è tipicamente qualcosa che hai già percorso — il telefono o l'orologio ha registrato ogni punto GPS lungo il cammino. Un percorso è solitamente pianificato in anticipo, collegando waypoint che rappresentano dove intendi andare.

Quando gli escursionisti usano le tracce GPX

Associazioni escursionistiche, guide e compagni di sentiero condividono tracce GPX così puoi seguire un percorso già tracciato attraverso boschi, terreno alpino o passeggiate costiere. Caricare il file sul telefono ti dà un riferimento visivo anche quando i segnali sul sentiero sono scarsi. Quando i ciclisti usano i percorsi GPX

I ciclisti pianificano anelli, gran fondo e avventure gravel come percorsi GPX, poi li caricano su telefono o ciclocomputer. Se vuoi crearne uno tuo, leggi come creare percorsi ciclistici GPX per un flusso di lavoro passo dopo passo.

Perché il formato file GPX è diventato lo standard

Prima del GPX, condividere dati GPS tra marchi diversi era complicato. Ogni produttore usava il proprio formato binario e convertire i file richiedeva software speciale. Il GPS Exchange Format ha risolto il problema pubblicando uno schema XML aperto che chiunque potesse implementare.

Domande frequenti sui file GPX

GPX è uguale a KML?

No. KML è un formato cartografico diverso. GPX è la scelta preferita per la navigazione GPS e il seguimento di percorsi outdoor. GPX Viewer apre file .gpx direttamente su iPhone e Android.

Quanto sono grandi i file GPX?

Una breve passeggiata nel quartiere può produrre un file da 50 KB, mentre un trekking di più giorni con campionamento GPS frequente può raggiungere diversi megabyte. I dati altimetrici e i timestamp aumentano le dimensioni, ma anche i file più grandi si aprono rapidamente nelle app mobili moderne.

Posso modificare un file GPX?

Sì. In GPX Viewer puoi creare percorsi da zero, modificare waypoint ed esportarli come nuovi file .gpx da condividere con amici o riaprire in seguito.
