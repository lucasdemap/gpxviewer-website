Come aprire file GPX su iPhone e Android

Importa percorsi GPX da Mail, File o Safari direttamente in GPX Viewer.

11 luglio 2026

Hai trovato un ottimo sentiero online, un amico ti ha inviato un anello ciclistico o hai esportato una registrazione GPS — ora devi aprire il file GPX sul telefono. La buona notizia: importare GPX su iPhone e Android è semplice una volta che sai quale menu di condivisione usare.

Questa guida illustra ogni percorso di importazione comune e mostra come GPX Viewer visualizza il tuo percorso su una mappa in pochi secondi. Se non conosci ancora il formato, inizia con cos'è un file GPX per capire tracce, percorsi e waypoint.

Prima di importare: installa un visualizzatore GPX

iOS e Android non includono un visualizzatore mappa GPX integrato. Ti serve un'app che comprenda il formato .gpx. GPX Viewer è scaricabile gratis su entrambe le piattaforme e gestisce importazione, visualizzazione mappa, profili altimetrici e creazione percorsi.

  • Scarica GPX Viewer da App Store o Google Play.
  • Apri l'app una volta così si registra come gestore GPX sul dispositivo.
  • Tieni il file GPX a portata di mano — in Mail, File o nella cartella download.

Come aprire un file GPX su iPhone

Il foglio di condivisione di Apple è il modo più veloce per importare GPX su iPhone. L'etichetta varia leggermente a seconda di dove si trova il file, ma il flusso è sempre: trova il file → tocca Condividi → scegli GPX Viewer.

Dall'app Mail

  1. Apri l'email con l'allegato .gpx.
  2. Tocca l'allegato per visualizzarlo in anteprima.
  3. Tocca l'icona di condivisione nell'angolo in basso a sinistra.
  4. Scorri la riga delle app e seleziona GPX Viewer.
  5. Il percorso appare subito sulla mappa.

Dall'app File

  1. Trova il file .gpx in iCloud Drive, Sul mio iPhone o un servizio cloud collegato.
  2. Tieni premuto il file e tocca Condividi, oppure aprilo e tocca l'icona di condivisione.
  3. Scegli GPX Viewer dall'elenco delle app.

Da Safari

Molti siti di sentieri offrono un link diretto per scaricare GPX. Tocca il link, poi Scarica. Quando il download è completato, tocalo nel gestore download di Safari e condividilo con GPX Viewer. Puoi anche salvare il file in File e importarlo da lì.

Da un file GPX scaricato

Se qualcuno ti ha inviato un percorso o hai salvato un .gpx da un sito web, apri il file e tocca Condividi. Scegli GPX Viewer dall'elenco. Il percorso appare subito sulla mappa — senza conversione o passaggi extra.

Come aprire un file GPX su Android

La finestra «Apri con» di Android funziona in modo simile alla condivisione iOS, anche se i menu appaiono diversi a seconda della personalizzazione del produttore.

Da Gmail o qualsiasi app email

  1. Apri il messaggio con l'allegato GPX.
  2. Tocca l'allegato per scaricarlo.
  3. Quando richiesto, scegli GPX Viewer come app di apertura.
  4. Se non appare alcun prompt, apri la cartella Download e tocca il file.

Da archivio locale o cartella cloud

  1. Trova il file .gpx nel gestore file del telefono o in una cartella cloud.
  2. Tocca il file, poi seleziona Apri con o il menu a tre puntini → Condividi.
  3. Scegli GPX Viewer dall'elenco delle app.

Dal browser mobile

Scarica il GPX da un sito web, poi apri l'elenco download del browser. Tocca il file completato e seleziona GPX Viewer. Se Android chiede se aprire il file solo una volta o sempre con GPX Viewer, scegli Sempre per importazioni più rapide in futuro.

Risoluzione dei problemi di importazione GPX

La maggior parte dei fallimenti di importazione deriva da poche cause prevedibili. Segui questa checklist prima di pensare che il file sia danneggiato:

  • Tipo di file errato — verifica che l'estensione sia .gpx, non .kml o .tcx.
  • Traccia vuota — alcuni file contengono metadati ma nessun punto GPS; prova a riesportare il percorso da GPX Viewer.
  • App non presente in Condividi — apri GPX Viewer prima, poi riprova; riavvia il telefono se necessario.
  • File molto grande — le registrazioni di più giorni con GPS ogni secondo possono essere lente; attendi qualche secondo per l'analisi.

Se il file si apre ma appare sbagliato sulla mappa, leggi la nostra guida su come visualizzare GPX su una mappa per verificare stile mappa, livello di zoom e impostazioni altimetriche.

Cosa succede dopo l'importazione?

Una volta che GPX Viewer carica il file, vedrai l'intero percorso disegnato su una mappa interattiva. Scorri verso l'alto o tocca la scheda del percorso per visualizzare distanza, dislivello e dettagli dei waypoint. Passa tra mappe standard, satellitare e ibrida a seconda che tu stia esplorando sentieri boschivi o strade aperte.

Con GPX Viewer Pro, puoi anche attivare la posizione in tempo reale per vedere posizione e direzione sovrapposte alla traccia — essenziale per seguire tracce GPX durante l'escursionismo o navigare un percorso ciclistico pianificato.

Organizzare più file GPX

GPX Viewer conserva i percorsi importati nella tua libreria così puoi passare da uno all'altro senza reimportare. Dai un nome ai percorsi quando li salvi, raggruppali per tipo di attività ed elimina i file vecchi che non ti servono più. Quando sei pronto a condividere un percorso, usa la funzione di esportazione per generare un nuovo file .gpx da inviare agli amici.
