Perguntas frequentes

Que tipos de ficheiro são suportados?

O GPX Viewer funciona com ficheiros .gpx — o formato padrão para rotas e trilhos GPS.

Existe subscrição?

Não. O GPX Viewer Pro é uma compra única. Compre uma vez, fique para sempre.

Posso abrir ficheiros GPX de outras apps?

Sim. Partilhe um ficheiro GPX para o GPX Viewer a partir de Ficheiros, Mail, Safari e outras apps.
