FAQ

Jenis file apa yang didukung?

GPX Viewer bekerja dengan file .gpx — format standar untuk rute dan track GPS.

Apakah ada langganan?

Tidak. GPX Viewer Pro adalah pembelian sekali bayar. Beli sekali, miliki selamanya.

Bisakah saya membuka file GPX dari aplikasi lain?

Ya. Bagikan file GPX ke GPX Viewer dari Files, Mail, Safari, dan aplikasi lain.

Butuh bantuan?

Siap menjelajahi rute berikutnya?
