Ringkasan

GPX Viewer (“aplikasi”) dikembangkan oleh tim GPX Viewer. Kebijakan ini menjelaskan informasi apa yang dikumpulkan aplikasi dan situs web serta bagaimana informasi tersebut digunakan.

Data aplikasi

Rute dan file GPX yang Anda impor tetap berada di perangkat Anda kecuali Anda memilih untuk mengekspor atau membagikannya. Data lokasi hanya digunakan saat Anda mengaktifkan fitur lokasi untuk menampilkan posisi Anda di peta.

Analitik

Aplikasi dan situs web menggunakan Google Analytics / Firebase Analytics untuk memahami penggunaan dan meningkatkan produk. Data analitik diagregasi dan tidak mencakup isi file GPX Anda.

Pembelian

Pembelian dalam aplikasi diproses oleh Apple dan Google. Kami tidak menerima detail kartu pembayaran Anda.
