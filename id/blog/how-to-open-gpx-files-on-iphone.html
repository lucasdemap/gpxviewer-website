Cara Membuka File GPX di iPhone dan Android

Impor rute GPX dari Mail, Files, atau Safari langsung ke GPX Viewer.

11 Juli 2026

Anda menemukan jalur bagus online, teman mengirim loop bersepeda, atau mengekspor rekaman GPS — sekarang Anda perlu membuka file GPX di ponsel. Kabar baiknya: mengimpor GPX di iPhone dan Android cukup mudah setelah tahu menu berbagi yang tepat.

Panduan ini membahas setiap jalur impor umum dan menunjukkan bagaimana GPX Viewer menampilkan rute di peta dalam hitungan detik. Jika Anda baru mengenal formatnya, mulai dengan apa itu file GPX untuk latar belakang tentang track, rute, dan waypoint.

Sebelum impor: pasang GPX viewer

iOS dan Android tidak dilengkapi penampil peta GPX bawaan. Anda memerlukan aplikasi yang memahami format .gpx. GPX Viewer gratis diunduh di kedua platform dan menangani impor, tampilan peta, grafik elevasi, serta pembuatan rute.

  • Unduh GPX Viewer dari App Store atau Google Play.
  • Buka aplikasi sekali agar terdaftar sebagai handler GPX di perangkat.
  • Siapkan file GPX — di Mail, Files, atau folder unduhan.

Cara membuka file GPX di iPhone

Share sheet Apple adalah cara tercepat mengimpor GPX di iPhone. Labelnya sedikit berbeda tergantung lokasi file, tetapi alurnya selalu: temukan file → ketuk Bagikan → pilih GPX Viewer.

Dari aplikasi Mail

  1. Buka email dengan lampiran .gpx.
  2. Ketuk lampiran untuk pratinjau.
  3. Ketuk ikon bagikan di sudut kiri bawah.
  4. Gulir baris aplikasi dan pilih GPX Viewer.
  5. Rute langsung muncul di peta.

Dari aplikasi Files

  1. Temukan file .gpx di iCloud Drive, Di iPhone Saya, atau layanan cloud terhubung.
  2. Tekan lama file dan ketuk Bagikan, atau buka dan ketuk ikon bagikan.
  3. Pilih GPX Viewer dari daftar aplikasi.

Dari Safari

Banyak situs trail menawarkan tautan unduhan GPX langsung. Ketuk tautan, lalu Unduh. Saat unduhan selesai, ketuk di manajer unduhan Safari dan bagikan ke GPX Viewer. Anda juga bisa menyimpan ke Files terlebih dahulu dan impor dari sana.

Dari file GPX yang diunduh

Jika seseorang mengirim rute atau Anda menyimpan .gpx dari situs web, buka file dan ketuk Bagikan. Pilih GPX Viewer dari daftar. Rute langsung muncul di peta — tanpa konversi atau langkah tambahan.

Cara membuka file GPX di Android

Dialog buka-dengan Android bekerja mirip berbagi iOS, meski menu terlihat berbeda tergantung skin produsen.

Dari Gmail atau aplikasi email

  1. Buka pesan dengan lampiran GPX.
  2. Ketuk lampiran untuk mengunduh.
  3. Saat diminta, pilih GPX Viewer sebagai aplikasi pembuka.
  4. Jika tidak muncul prompt, buka folder Unduhan dan ketuk file.

Dari penyimpanan lokal atau folder cloud

  1. Temukan file .gpx di file manager ponsel atau folder cloud.
  2. Ketuk file, lalu pilih Buka dengan atau menu tiga titik → Bagikan.
  3. Pilih GPX Viewer dari daftar aplikasi.

Dari browser mobile

Unduh GPX dari situs web, lalu buka daftar unduhan browser. Ketuk file selesai dan pilih GPX Viewer. Jika Android menanyakan buka sekali atau selalu dengan GPX Viewer, pilih Selalu untuk impor lebih cepat ke depannya.

Mengatasi masalah impor GPX

Sebagian besar kegagalan impor berasal dari beberapa penyebab yang dapat diprediksi. Periksa daftar ini sebelum menganggap file rusak:

  • Jenis file salah — pastikan ekstensinya .gpx, bukan .kml atau .tcx.
  • Track kosong — beberapa file berisi metadata tanpa titik GPS; coba ekspor ulang dari GPX Viewer.
  • Aplikasi tidak ada di Bagikan — buka GPX Viewer dulu, lalu coba lagi; restart ponsel jika perlu.
  • File sangat besar — rekaman multi-hari dengan GPS per detik bisa lambat; tunggu beberapa detik untuk parsing.

Jika file terbuka tetapi terlihat salah di peta, baca panduan kami tentang cara melihat GPX di peta untuk memeriksa gaya peta, zoom, dan pengaturan elevasi.

Apa yang terjadi setelah impor?

Setelah GPX Viewer memuat file, Anda melihat rute penuh digambar di peta interaktif. Geser ke atas atau ketuk kartu rute untuk melihat jarak, kenaikan elevasi, dan detail waypoint. Beralih antara lapisan peta standar, satelit, dan hybrid tergantung apakah Anda menelusuri jalur hutan atau jalan terbuka.

Dengan GPX Viewer Pro, Anda juga bisa mengaktifkan lokasi langsung untuk melihat posisi dan arah real-time di atas track — penting untuk mengikuti jalur GPX saat hiking atau menavigasi rute bersepeda yang direncanakan.

Mengorganisir banyak file GPX

GPX Viewer menyimpan rute yang diimpor di perpustakaan agar Anda bisa beralih tanpa impor ulang. Beri nama rute saat menyimpan, kelompokkan menurut jenis aktivitas, dan hapus file lama yang tidak diperlukan. Saat siap membagikan kembali, gunakan fitur ekspor untuk menghasilkan file .gpx baru untuk teman.
