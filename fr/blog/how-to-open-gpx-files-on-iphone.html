Comment ouvrir des fichiers GPX sur iPhone et Android

Importez des itinéraires GPX depuis Mail, Fichiers ou Safari directement dans GPX Viewer.

11 juillet 2026

Vous avez trouvé un super sentier en ligne, un ami vous a envoyé une boucle vélo ou vous avez exporté un enregistrement GPS — il vous faut maintenant ouvrir le fichier GPX sur votre téléphone. La bonne nouvelle : importer un GPX sur iPhone et Android est simple une fois que vous savez quel menu de partage utiliser.

Ce guide parcourt chaque méthode d'import courante et montre comment GPX Viewer affiche votre itinéraire sur une carte en quelques secondes. Si le format vous est inconnu, commencez par qu'est-ce qu'un fichier GPX pour comprendre traces, itinéraires et points de passage.

Avant l'import : installez un visualiseur GPX

iOS et Android ne proposent pas de visualiseur GPX intégré. Vous avez besoin d'une application qui comprend le format .gpx. GPX Viewer est gratuit sur les deux plateformes et gère l'import, l'affichage cartographique, les profils d'altitude et la création d'itinéraires.

  • Téléchargez GPX Viewer depuis l'App Store ou Google Play.
  • Ouvrez l'application une fois pour qu'elle s'enregistre comme gestionnaire GPX sur votre appareil.
  • Gardez le fichier GPX à portée de main — dans Mail, Fichiers ou votre dossier de téléchargements.

Comment ouvrir un fichier GPX sur iPhone

La feuille de partage d'Apple est le moyen le plus rapide d'importer un GPX sur iPhone. Le libellé exact varie légèrement selon l'emplacement du fichier, mais le flux est toujours : trouver le fichier → appuyer sur Partager → choisir GPX Viewer.

Depuis l'application Mail

  1. Ouvrez l'e-mail contenant la pièce jointe .gpx.
  2. Appuyez sur la pièce jointe pour la prévisualiser.
  3. Appuyez sur l'icône de partage en bas à gauche.
  4. Faites défiler la rangée d'applications et sélectionnez GPX Viewer.
  5. L'itinéraire apparaît immédiatement sur la carte.

Depuis l'application Fichiers

  1. Localisez le fichier .gpx dans iCloud Drive, Sur mon iPhone ou un service cloud connecté.
  2. Appuyez longuement sur le fichier et touchez Partager, ou ouvrez-le et appuyez sur l'icône de partage.
  3. Choisissez GPX Viewer dans la liste des applications.

Depuis Safari

De nombreux sites de sentiers proposent un lien de téléchargement GPX direct. Appuyez sur le lien, puis sur Télécharger. Une fois le téléchargement terminé, appuyez dessus dans le gestionnaire de téléchargements de Safari et partagez-le vers GPX Viewer. Vous pouvez aussi enregistrer le fichier dans Fichiers et l'importer depuis là.

Depuis un fichier GPX téléchargé

Si quelqu'un vous a envoyé un itinéraire ou si vous avez enregistré un .gpx depuis un site web, ouvrez le fichier et appuyez sur Partager. Choisissez GPX Viewer dans la liste. L'itinéraire apparaît immédiatement sur la carte — sans conversion ni étape supplémentaire.

Comment ouvrir un fichier GPX sur Android

La boîte de dialogue « Ouvrir avec » d'Android fonctionne de manière similaire au partage iOS, bien que les menus diffèrent selon la surcouche du fabricant.

Depuis Gmail ou toute application e-mail

  1. Ouvrez le message contenant la pièce jointe GPX.
  2. Appuyez sur la pièce jointe pour la télécharger.
  3. Lorsque vous y êtes invité, choisissez GPX Viewer comme application d'ouverture.
  4. Si aucune invite n'apparaît, ouvrez votre dossier Téléchargements et appuyez sur le fichier.

Depuis le stockage local ou un dossier cloud

  1. Trouvez le fichier .gpx dans le gestionnaire de fichiers de votre téléphone ou un dossier cloud.
  2. Appuyez sur le fichier, puis sélectionnez Ouvrir avec ou le menu trois points → Partager.
  3. Choisissez GPX Viewer dans la liste des applications.

Depuis votre navigateur mobile

Téléchargez le GPX depuis un site web, puis ouvrez la liste des téléchargements de votre navigateur. Appuyez sur le fichier terminé et sélectionnez GPX Viewer. Si Android demande d'ouvrir le fichier une seule fois ou toujours avec GPX Viewer, choisissez Toujours pour des imports plus rapides par la suite.

Résoudre les problèmes d'import GPX

La plupart des échecs d'import viennent de quelques causes prévisibles. Parcourez cette liste avant de supposer que le fichier est corrompu :

  • Mauvais type de fichier — confirmez que l'extension est .gpx, pas .kml ou .tcx.
  • Trace vide — certains fichiers contiennent des métadonnées mais aucun point GPS ; essayez de réexporter l'itinéraire depuis GPX Viewer.
  • Application absente du menu Partager — ouvrez GPX Viewer d'abord, puis réessayez ; redémarrez le téléphone si nécessaire.
  • Fichier très volumineux — les enregistrements sur plusieurs jours avec un GPS échantillonné chaque seconde peuvent être lents ; attendez quelques secondes pour l'analyse.

Si le fichier s'ouvre mais semble incorrect sur la carte, consultez notre guide sur comment afficher un GPX sur une carte pour vérifier le style de carte, le niveau de zoom et les paramètres d'altitude.

Que se passe-t-il après l'import ?

Une fois GPX Viewer chargé votre fichier, vous verrez l'itinéraire complet tracé sur une carte interactive. Balayez vers le haut ou appuyez sur la fiche d'itinéraire pour voir la distance, le dénivelé et les détails des points de passage. Basculez entre les couches standard, satellite et hybride selon que vous explorez des sentiers forestiers ou des routes ouvertes.

Avec GPX Viewer Pro, vous pouvez aussi activer la position en direct pour voir votre position et orientation en temps réel superposées à la trace — essentiel pour suivre des sentiers GPX en randonnée ou naviguer un itinéraire vélo planifié.

Organiser plusieurs fichiers GPX

GPX Viewer conserve les itinéraires importés dans votre bibliothèque pour basculer entre eux sans réimporter. Nommez les itinéraires lors de l'enregistrement, regroupez-les par type d'activité et supprimez les anciens fichiers dont vous n'avez plus besoin. Quand vous êtes prêt à partager un itinéraire, utilisez la fonction d'export pour générer un nouveau fichier .gpx à envoyer à vos amis.
