Téléchargez GPX depuis Strava

Exportez les activités et ouvrez-les dans GPX Viewer.

25 juillet 2026

Sur Strava web : ouvrez une activité → menu ⋮ → Export GPX. Enregistrez le fichier, puis ouvrez-le dans GPX Viewer sur votre téléphone.

Conseils

  • Certaines activités peuvent être restreintes à l'export par le propriétaire
  • Le GPX inclut la trace — vérifiez l'altitude avant de rouler
  • Partagez le même fichier avec votre groupe via le partage multiplateforme
