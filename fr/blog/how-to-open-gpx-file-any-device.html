Comment ouvrir un fichier GPX sur n'importe quel appareil

Téléphones, ordinateurs et unités GPS : ce qui ouvre les fichiers .gpx.

14 juillet 2026

GPX est un standard ouvert — de nombreuses applications le lisent. Sur iPhone et Android, GPX Viewer est le plus rapide : partagez le fichier vers l'app et l'itinéraire apparaît immédiatement.

Par appareil

  • iPhone / Android — GPX Viewer (recommandé) ou import via le menu Partager
  • Mac / Windows — Garmin BaseCamp, QGIS ou Google Earth
  • Montres GPS — import via Garmin Connect ou l'app du fabricant
